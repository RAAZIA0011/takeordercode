## ☕ Brew Haven Café — C++ Console Application

**Brew Haven** is a console-based café ordering system built in C++ as a collaborative academic project. It simulates a real café experience right in the terminal — from an animated welcome screen to a full order summary with discounts.

### Features
- 🎬 Animated loading screen and ASCII café banner on startup
- 📋 Full menu across 4 categories: Coffee, Fast Food, Cold Drinks, and Desserts (21 items total)
- ✅ Robust input validation with looping checks and error messages
- 🛒 Multi-item ordering session — add items from any category before checkout
- 🧾 Detailed order summary with per-item breakdown and total
- 💸 Automatic discount system:
  - 5% off on bills ≥ Rs. 1,000
  - 10% off on bills ≥ Rs. 2,000
  - 15% off on bills ≥ Rs. 3,000

### Tech Stack
- Language: C++
- Concepts used: arrays, functions, loops, input validation, string handling, console formatting

### Project Structure
This project was developed collaboratively. My responsibility was the **order-taking module** — including category order functions, input validation logic, order storage, and the billing summary.
